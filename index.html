About Me

어떤 개발자인가요?

🎮 데스 런 (Death Run)
완료

플레이어가 다양한 장애물을 피해 목적지까지 도달하는 로블록스 게임입니다. 기본 이동 속도 120(21.6 km/h)으로 빠르게 장애물을 돌파하며, 포털을 통해 총 6개의 스테이지를 순서대로 클리어합니다.

▶ 장애물: 50×50 정사각형 데스 박스, 60×20 직사각형 데스 박스, 하늘에서 떨어지는 원형 장애물, 구체를 발사하는 추적 몬스터
▶ 산데비스탄 (쿨타임 5초): 주변 장애물 속도를 5초간 50으로 감속 — 회피 타이밍 확보
▶ 충격파: 전방의 적·장애물을 멀리 밀쳐내는 스킬
▶ 포털을 통한 6스테이지 구성

🛠️ SQL Generator
완료 (Alpha)

자연어를 입력하면 SQL 쿼리를 즉시 생성해주는 개발자 도구입니다. SQL 학습 중 모르는 문법이 생길 때마다 반복되는 검색 → 결과 확인 → 판별 과정을 없애고, 말하듯 입력하면 바로 완성된 쿼리를 받을 수 있습니다. Claude API(claude-sonnet-4-6) 기반으로 동작합니다.

▶ 자연어 모드: 한국어 입력 지원, 한글 테이블·컬럼명 자동 영문 변환 (인간 → human, 나이 → age)
▶ 스키마 모드: 실제 DDL을 입력해 스키마에 맞는 정밀한 쿼리 생성 (직접 입력 / 예시 / .sql 파일 업로드)
▶ 5종 DB 지원: MySQL · MariaDB · PostgreSQL · SQLite · SQL Server · Oracle — DB별 전용 문법으로 최적화
▶ 결과 출력: 신택스 하이라이팅 + 한국어 설명 + 팁 & 주의사항 + 클립보드 복사
▶ 세션 내 최근 10개 쿼리 기록 · ⌘+Enter 단축키 · HP Design System 적용 UI
